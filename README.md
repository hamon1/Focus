# 🐱 Focus Timer (Electron 기반 데스크톱 타이머 앱)

## 1. 프로젝트 소개

사용자의 집중 시간을 효율적으로 관리하고, 지속적인 사용을 유도하기 위해 설계한 **데스크톱 뽀모도로 타이머 애플리케이션**입니다.
단순 타이머 기능을 넘어서 **목표 관리, 세션 기록, 인터랙션 요소(UI/UX)** 를 결합하여 몰입 경험을 개선하는 것을 목표로 개발했습니다.

---

## 2. 개발 의도

### ✔ 문제 인식

* 웹 기반 타이머는 백그라운드 동작이 불안정하고 접근성이 낮음
* 기존 타이머 앱은 단순 반복 구조로 인해 지속 사용 유도가 어려움

### ✔ 해결 전략

* 항상 화면 위에 유지되는 **데스크톱 타이머**
* 최소 UI 방해를 위한 **투명/미니 모드**
* 사용자 행동을 유도하는 **인터랙션 요소 (밈 팝업)**

---

## 3. 주요 기능

### ⏱ 타이머

* Start / Pause / Stop / Reset 상태 관리
* 스톱워치 기반 구조 (확장 가능한 설계)
* 세션 종료 시 자동 기록

### 🎯 목표 관리

* 목표(goal) 추가 및 선택
* 현재 작업 표시
* 목표별 집중 시간 기록

### 📊 기록 (History)

* 완료된 세션 기록 누적
* 시간 + 목표 기반 로그 관리

### 🖥 UI/UX

* **3단계 화면 모드**

  * Mini / Timer / Full
* 투명 창 + 드래그 영역 커스터마이징
* 중앙 정렬 및 반응형 UI
* Light / Dark 테마 지원

### 🐾 인터랙션

* 타이머 이벤트 기반 **고양이 밈 팝업**
* 별도 창을 활용한 오버레이 효과

### ⚙️ 사용자 설정

* 투명도 조절 (슬라이더)
* 설정값 로컬 저장

## 4. 기술 스택

* Electron
* Vanilla JavaScript
* HTML / CSS
* electron-store

---

## 5. 아키텍처

```
Renderer (UI)
   ↓
Preload (contextBridge)
   ↓
Main Process (IPC)
   ↓
OS 기능 (Window 제어, 로컬 저장)
```

### ✔ 설계 포인트

* `contextIsolation` 환경에서 안전한 API 노출
* Renderer와 Main 역할 분리
* IPC 기반 데이터 흐름 설계

---

## 6. 기술적 의사결정

### 1) 타이머 구조

* `elapsed` 기반 설계
* 기록 중심 기능에 적합
* countdown 방식으로 확장 가능

### 2) UI 상태 관리

* `body class` 기반 상태 제어 (mini / timer / full)
* 로직과 스타일 분리

### 3) 투명도 제어

* CSS 변수(`--opacity`) 사용
* 전역 상태 일원화

### 4) 팝업 시스템

* DOM이 아닌 **별도 BrowserWindow 사용**
* 윈도우 영역 밖 표현 문제 해결

## 8. 트러블 슈팅

### ✔ transform vs layout 문제

* `scale` 사용 시 위치가 어긋나는 문제 발생
* 해결: width/height 기반으로 크기 제어

### ✔ canvas 중앙 정렬

* flex 레이아웃 미적용으로 위치 오류 발생
* 해결: 부모 컨테이너 기준 중앙 정렬

### ✔ IPC 통신 문제

* Renderer에서 직접 Electron API 접근 불가
* 해결: preload를 통한 안전한 bridge 구성
